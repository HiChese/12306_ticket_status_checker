# 12306 车票查询工具 (尚在开发UI中)

基于 Python 爬虫引擎 + C++ 交互界面的 12306 实时余票查询工具。无需浏览器，纯 HTTP 请求，自动防爬虫。

## 功能

- 按城市或精确站名查询任意区间列车时刻表
- 显示同城所有车站（如输入"北京"自动包含北京站、北京南、北京西、北京丰台等）
- 实时展示各席位余票（商务座/一等座/二等座/动卧/软卧/硬卧/硬座/无座等）
- 支持 G/C/D/K/T/Z 及纯数字普速列车
- 查询结果自动保存为 JSON 文件
- 多次连续查询，无需重启

## 快速开始

### 下载运行 (尚在开发UI中)

从 [Releases](../../releases) 下载 `12306查询.exe`，双击运行即可。

### 开发运行

```bash
pip install -r requirements.txt
python main.py
```

## 席位解析原理

12306 返回的每条列车数据是一串 `|` 分隔的 58 个字段：

```
[3]=车次  [6]=上车站  [7]=下车站  [8]=发时  [9]=到时  [10]=历时
[26]=无座  [28-32]=其他席位（位置因 G/D/K 车类型而异）
```

程序按车次首字母选用席位偏移表，从对应位置取出余票值，再用 `data.map` 将电报码翻译为中文站名。全程纯规则解析，无 AI 参与。
